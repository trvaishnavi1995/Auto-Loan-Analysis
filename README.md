# Auto-Loan-Analysis
# 📊 Auto Loan Analysis Dashboard (Power BI)

## 📌 Overview
This project analyzes auto loan origination trends using data from CFPB. The goal is to uncover patterns in loan growth, seasonal behavior, and year-over-year performance.

---

## 📂 Dataset
- Source: Consumer Financial Protection Bureau (CFPB)
- Data includes:
  - Monthly auto loan originations
  - Seasonally adjusted & unadjusted values
  - Time range: 2006–2025

---

## 🛠 Tools Used
- Power BI (Dashboard & Visualization)
- Excel (Data Cleaning & Preparation)

---

## 📊 Key Features

### 🔹 KPI Metrics
- Total Originations
- Average Monthly Originations
- Peak & Minimum values

### 🔹 Visualizations
- Monthly Trend Analysis (Line Chart)
- Yearly Comparison (Bar Chart)
- Year-over-Year Growth (YoY %)
- Interactive Slicer (Year filter)

---

## 📈 Insights
- Significant drop observed during financial crisis period
- Strong recovery phase post-2010
- Noticeable fluctuation during COVID years
- YoY growth highlights volatility in recent years
